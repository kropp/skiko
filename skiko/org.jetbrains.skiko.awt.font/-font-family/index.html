FontFamily

class FontFamily(val familyName: String, val source: FontFamily.FontFamilySource, typefacesByStyle: MutableMap<FontStyle, Typeface> = mutableMapOf()) : Map<FontStyle, Typeface>

Holds info about a font family. Intended to be used as a cache entry, to which typefaces are lazily loaded as new Typefaces are loaded.

See also

familyName

The primary font family name. Must not be blank.
