fun fillPathWithPaint(src: Path, paint: Paint, cull: Rect?, resScale: Float): Path

Returns the filled equivalent of the stroked path using the provided paint attributes.

Return

A filled version of the source path.

Parameters

src

Path to create a filled version of.

paint

Paint from which attributes such as stroke cap, width, miter, join, and pathEffect will be used.

cull

Optional limit passed to the path effect.

resScale

If > 1, increase precision, else if (0 < resScale < 1) reduce precision to favor speed and size.
